React-basic projesinde veriler dummy olarak alınmıştır. Show persons butonuna basınca kişiler isim ve yaş bilgileriyle ekrana gelmektedir.
Input kısmındaki isim değişince otomatik olarak bilgi değişmektedir. Ayrıca kişinin üzerine tıklayınca kişi silinmektedir.
Toggle persons butonuna tıklayınca kişi bilgileri tekrardan gizlenmektedir.
